# 혜달이의 상태 (Hye-Otter)

> 귀여운 해달 캐릭터 '혜달이'로 나의 현재 상태를 연인/가족에게 공유하는 타이머 웹앱

## 컨셉

혜달이는 단순한 상태 알림이 아닙니다.
**다마고치**처럼 함께 키우고, **뽀모도로 타이머**로 집중을 돕고, 사랑하는 사람에게 **지금 나의 상태**를 귀엽게 보여주는 앱입니다.

---

## 설계 단계

### 1단계: 프로젝트 기초 세팅
- 폴더 구조 생성 (`index.html`, `css/`, `js/`, `assets/`)
- `.gitignore` 설정
- 기본 HTML 뼈대 작성

### 2단계: 혜달이 SVG 캐릭터 제작
- 기본 해달 캐릭터 SVG 디자인
- 상태별 표정/포즈 변형 (집중, 휴식, 행복, 피곤, 배고픔 등)
- CSS 애니메이션 (흔들기, 반짝이기, 떠다니기)

### 3단계: 메인 UI 레이아웃
- 모바일 우선 반응형 디자인
- 혜달이가 중앙에 위치한 메인 화면
- 상태 패널, 타이머 패널, 다마고치 패널 구성
- 따뜻한 색감의 테마 스타일링

### 4단계: 뽀모도로 타이머
- 25분 집중 / 5분 휴식 기본 타이머
- 타이머 상태에 따라 혜달이 표정 변화
- 시작/일시정지/리셋 컨트롤
- 알림음 재생

### 5단계: 기분/감정 상태
- 감정 선택 버튼 (행복, 집중, 피곤, 졸림, 신남 등)
- 선택한 감정에 따라 혜달이 표정 + 배경색 변화
- 한줄 상태 메시지 입력

### 6단계: 다마고치 요소
- 혜달이 스탯: 배고픔, 행복도, 에너지
- 밥 주기 / 놀아주기 / 재우기 상호작용 버튼
- 시간이 지나면 스탯이 자연 감소
- 스탯에 따른 혜달이 상태 자동 변화
- `localStorage`로 데이터 저장

### 7단계: 상태 공유 기능
- 현재 상태를 URL 파라미터로 인코딩
- 공유 링크 생성 버튼 (링크 복사)
- 공유 링크를 열면 읽기 전용 뷰로 혜달이 상태 확인
- 카카오톡 등에서 미리보기 지원 (OG 태그)

## 기술 스택

| 구분 | 선택 |
|------|------|
| 플랫폼 | 웹앱 (순수 HTML/CSS/JS) |
| 캐릭터 | SVG 일러스트 + CSS 애니메이션 |
| 데이터 저장 | localStorage |
| 상태 공유 | URL 파라미터 인코딩 |
| 배포 | GitHub Pages |

## 폴더 구조 (예정)

```
hye-otter/
├── index.html          # 메인 페이지
├── share.html          # 공유 링크용 읽기 전용 페이지
├── css/
│   └── style.css       # 전체 스타일
├── js/
│   ├── app.js          # 메인 앱 로직
│   ├── timer.js        # 뽀모도로 타이머
│   ├── tamagotchi.js   # 다마고치 시스템
│   └── share.js        # 상태 공유/인코딩
└── assets/
    └── otter.svg       # 혜달이 SVG 캐릭터
```
